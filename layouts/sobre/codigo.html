La tecnología tiene dos modos de funcionamiento:
AUTOMÁTICO: Interactúa con las energías del entorno generando partículas cuánticas libres.
ENFOCADO: El usuario utiliza un código para dirigir la resonancia hacia un fin específico.

El uso de códigos es un ejercicio mental que no requiere tecnología adicional.
El usuario solo necesita vincular los conceptos del código con el problema que está tratando de resolver.
Cuanto más amplio sea el pensamiento detrás del código, más posibilidades de ver un efecto.

Para construir un código es necesario vincular los conceptos del esquema formando una cadena cerrada.
Se deben respetar las líneas de conexión ya que no todos los conceptos están relacionados entre sí.
Los códigos se pueden aplicar tanto al usuario del dispositivo como a otra persona con la que tienen un estrecho vínculo mental.

Cuando el usuario envía un código, se genera una resonancia entre el biocampo receptor y el mensaje transmitido por la señal.
La respuesta generalmente se percibe como un 'efecto mariposa' y no como la aceptación de una orden.

  • FUNCIÓN
  • TRANSFORMACIÓN
  • REACCIÓN
  • NEO
  • TRABAJO
  • MEMORIA

FUNCIÓN


Representa la acción que queremos lograr con el código.

Como una ecuación matemática, depende de otros conceptos para completar los operandos y obtener el resultado.

TRANSFORMACIÓN


Se aplica cuando es necesario cambiar algún elemento de la situación.

A diferencia del trabajo, es un proceso natural que se desencadena por la voluntad propia del individuo.

REACCIÓN


Cada modificación inducida por los códigos genera un porcentaje de rechazo por parte del biocampo.

La inclusión del concepto de reacción en el código reduce este rechazo y aumenta las posibilidades de éxito.

NEO


Son los productos que resultan de nuevas combinaciones con el medio ambiente.

Se aplica cuando los problemas no se pueden resolver modificando elementos existentes. Por ejemplo, cuando se busca generar nuevas ideas.

TRABAJO


Representa el trabajo físico o mental que el individuo debe realizar para lograr el objetivo.

Se relaciona con la transformación de ideas en fuerzas capaces de modificar el entorno.

MEMORIA


Es responsable de almacenar elementos físicos (grasa) y abstractos (experiencias) dentro del cuerpo.

Debe incluirse en su código para que siga siendo efectivo sin la necesidad de implementarlo a diario.
